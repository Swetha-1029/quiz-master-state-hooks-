## Project Scope

A fully interactive quiz application built with React.js using State Hooks. 
Users can select a category (JavaScript, React, General Web), answer 15 multiple choice questions, and receive instant feedback on each answer. 
The app tracks the score in real time, displays a results screen with performance feedback, and allows users to review all answers at the end.
The project demonstrates core React concepts including useState, useCallback, custom hooks, conditional rendering, and component composition.

---

## Features

- Category selection: All, JavaScript, React, General Web
- 15 multiple choice questions with difficulty tags (Easy / Medium / Hard)
- Instant answer feedback — correct answer highlights green, wrong highlights red
- Real-time score and progress tracking
- Results screen with score percentage and performance message
- Review all answers after quiz ends
- Retry or go back to home

## Tech Stack

- React 18
- Vite
- CSS Modules
- React Hooks (useState, useCallback, custom hook)
